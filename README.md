# Markdown Preview App (Vue 3 + TypeScript)

A Markdown Preview application built with Vue 3, TypeScript, and Vite, converted from a React + TypeScript implementation.
It allows users to write Markdown on one side and preview the rendered output in real time.

# Features
	•Live Markdown editing and preview
	•Autosave content to localStorage
	•Load Markdown files (.md)
	•Download Markdown as a file
	•Save Markdown content to a mock API
	•Syntax highlighting for code blocks
	•Error Boundary handling
	•Fully responsive layout 

# Tech Stack
	•Vue 3
	•TypeScript
	•Vite
	•vue Router
	•Marked (Markdown parser)
	•Highlight.js (code syntax highlighting)
	•CSS Grid & Flexbox
